class LogCatSeedling(treatAsAndroidN: Boolean, logDelegate: LogDelegate)
Content copied to clipboard

General purpose logging for Android applications with auto tagging and long log wrapping. Implementation avoids the log size hard limit of Android and uses and efficient right index lookup to attempt splits on existing new lines before forcing hard breaks. Tagging is automatically implemented by using an exception stack trace to determine the calling code.
